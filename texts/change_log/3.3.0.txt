## Версия: 3.3.0 Release
## Статус: НЕ АКТУАЛЬНАЯ!
## Дата релиза: 02.10.24

### Добавлено:
- **Поддержка интернет-поиска** с использованием библиотеки `duckduckgo_search`. Теперь вы можете искать:
  - Веб-страницы, он же поиск
  - Изображения
  - Видео

- **Новая команда `/asklong`**. Эта команда позволяет отправлять более длинные запросы через файл, что помогает обойти ограничение Discord в 2000 символов. Особенно полезно для работы с кодом.

### Обновлено:
- Команда `/history` теперь отображает результаты открыто для вас. Текст и файлы больше не исчезают, что делает использование более удобным.
- При генерации изображений теперь отображается название модели ИИ, которая создала изображение, вместо системного названия.

### Удалено:
- Необязательный параметр `additionalmessage` для команды `/ask` так как теперь есть `/asklong`

## Версия: 4.5.0 Release
## Статус: Актуальная :white_check_mark:
## Дата релиза: 16.03.25

:rocket: **ГЛОБАЛЬНАЯ ПЕРЕРАБОТКА КОДА** :rocket:

### Добавлено:
- Динамический TTL кэш для повышения производительности и стабильности бота
- Thinking/Reasoning Embed ответы с поддержкой `send_split_message` и `send_split_code`
- Новые ИИ модели:
	1. `claude-3.7-sonnet`
	2. `qwen-2.5-1m-demo`
	3. `mixtral-small-24b`

### Улучшено/Исправлено:
- **pdfminer.six** заменен на **pyMuPDF**
- Рефакторинг обработки сообщений/команд:
	1. Удален `process_message`. Запросы выполняются напрямую
	2. Логика `handle_response` разделена на несколько функций
- Рефакторинг Админ функций:
	1. Добавлен `/ban-info` для проверки статуса бана
	2. Поддержка Discord Embed
	3. Цикл для авторазбана по истечении времени
- Рефакторинг Напоминаний:
	1. Улучшен алгоритм времени
	2. Уникальные ID для избежания конфликтов
- Рефакторинг Интернет-поиске:
	1. Использование одной сессии для запросов.
	2. Удалено дублирование кода
	3. Поддержка HTTP статус-кодов
	4. Новые проверки для повышения стабильности

### Обновлено:
- G4F обновлен до версии 0.4.8.6.
- Модели, провайдеры, библиотеки обновлены

### Удалено:
- Нестандартный RetryProvider. Теперь используется только G4F класс
